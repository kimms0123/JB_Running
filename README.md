# 중부런
오합지졸 게임 모작 프로젝트
---
## 📌 개요
- 프로젝트명: 중부런(JB-Running)
- 장르: 2D 러너 게임
- 개발 툴: Unity
- 개발 인원
  - 아트부: 유다현, 이지원, 김지우
  - 개발부: 김민선, 김은석, 이수안
  - 기획부: 민채빈
- 제작 기간: 2025.09.07 ~ 2025.11.03
---
## 📌 게임 설명
플레이어는 캐릭터를 조종해 끊임없이 앞으로 달려나가는 러너 게임입니다.
장애물을 피하고 아이템을 먹으며 최대한 멀리 달리는 것이 목표입니다.
장애물에 3번 이상 부딪히면 게임 오버가 되며, 플레이어의 최종 이동 거리가 표시됩니다.

## 📌 게임 흐름
1. 게임 시작
   - 플레이어 자동 이동
   - 배경 및 플랫폼 스크롤
2. 플레이 진행
   - 점프, 장애물 피하기, 아이템 획득
4. 피격 처리
    - 장애물 충돌 시 넉백 → 3회 누적 시 게임 오버
6. 게임 오버
    - 시간 정지(Time.timeScale = 0)
    - 게임 오버 패널 표시
    - 최종 거리 출력 + retry 버튼 활성화
8. Retry
   - retry 버튼 클릭 시 RestartGame() 씬 재시작
